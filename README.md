# Campus Katale - UI Polish

This repository contains UI polishes and enhancements to the Campus Katale platform, designed and implemented by the KCF Frontend students of Cohort 9, 2025.

## About

Campus Katale is a campus marketplace platform that connects students to buy and sell items within their campus community. This project focuses on improving the user interface and user experience of the platform through various UI enhancements and polish work.

## Technologies Used

- React + Vite
- Tailwind CSS
- React Router DOM
- Clerk (Authentication)
- Tabler Icons

## Features

- Product listings and search functionality
- Wishlist management
- Notification system
- User authentication
- Responsive design
- Modern UI components

## Development

This project was created using Vite and React, providing a fast development experience with Hot Module Replacement (HMR).
